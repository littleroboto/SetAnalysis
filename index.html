SetAnalysis Helper

Technical utility for set-overlap analysis where datasets are similar but not identical.

Open Workbench

Find critical differences, not averages

For investigating large collections of records that mostly look alike but aren't quite the same — store estates, product catalogues, customer profiles, configuration files. Where a headline average smooths the small-but-material differences away, this tool lays them out side by side so you can see clearly what's shared, what diverges, and which exceptions are worth your attention.

  • See where your datasets really overlap — and where they quietly diverge in ways an average would never show.
  • Explore the long tail: drill into the rare combinations that turn out to matter, or filter them out so the main signal stays clear.
  • Export each chart as a clean SVG, ready for reports or decks.

Operational notes

This utility runs fully client-side. Input data stays in-browser. The optional BYOK analysis controls are hidden unless explicitly enabled via query parameter.
