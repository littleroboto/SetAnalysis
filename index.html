SetAnalysis Workbench

Technical utility for set-overlap analysis where datasets are similar but not identical.

Open Workbench

Analyze overlap as structure, not averages

SetAnalysis focuses on intersections, concentration, and archetype patterns in high-dimensional set data. It is designed for cases where entities share many common elements but still diverge in meaningful edge combinations.

  • Single and dual dimension set inputs.
  • Trim controls for high-cardinality intersections.
  • Deterministic SVG export with evidence-first captions.
Open Workbench

Operational notes

This utility runs fully client-side. Input data stays in-browser. The optional BYOK analysis controls are hidden unless explicitly enabled via query parameter.
